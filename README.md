interactive-php
===============

CLI for php.


Test for debugger.php

- src/debugger.php

! 適宜、sudoをつける

Ubuntu installation:

* apt-get install php-pear
* pear channel-discover pear.phpunit.de
* pear channel-discover pear.symfony.com
* pear install -a pear.phpunit.de/PHPUnit

* phpunit [enter]
  で、コマンドのヘルプが出たらOK
